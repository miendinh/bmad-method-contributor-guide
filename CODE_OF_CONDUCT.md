# Code of Conduct

> ⚠️ This code of conduct applies to **this unofficial documentation repository only**, not to official BMAD-METHOD. For BMad's code of conduct, see: <https://github.com/bmad-code-org/BMAD-METHOD/blob/main/.github/CODE_OF_CONDUCT.md>

---

## Our pledge

We pledge to create a welcoming, inclusive, and harassment-free environment for everyone — regardless of age, body size, disability, ethnicity, gender identity, experience level, nationality, race, religion, sexual orientation.

## Our standards

### ✅ Expected behavior

- Respectful, constructive communication
- Focus on what's best for the community
- Accept constructive criticism gracefully
- Show empathy toward other community members
- Honor attribution and acknowledgment

### ❌ Unacceptable behavior

- Harassment, trolling, personal attacks
- Discriminatory language or jokes
- Publishing others' private information
- Sexual attention or advances (unwelcome)
- Other conduct reasonably considered inappropriate

### 🚫 Specific to this repo

- False claims about official BMad endorsement
- Trademark violations (using "BMad" in product names)
- Spreading misinformation about BMAD-METHOD
- Spam, self-promotion for unrelated products

## Enforcement

Violations may result in:

1. **Warning** — Private message explaining issue
2. **Temporary ban** — Limited interaction for set period
3. **Permanent ban** — Complete removal from community

## Reporting

Report violations via:
- This repository's Issues tab (public)
- Repository maintainer's contact (private)

Reports handled confidentially. Retaliation against reporters is not tolerated.
